# Emergency Stop for Your AI Agents

$KILLSWITCH is an open-source safety layer for AI agents. Think of it as a circuit breaker that prevents AI from causing damage.

## The Problem

AI agents can:
- Delete production databases
- Spend thousands on API calls
- Leak sensitive credentials
- Execute dangerous commands
- Transfer funds without approval

And they do all this in milliseconds.

## The Solution

$KILLSWITCH validates every action before it happens:

1. **Real-time Validation** - Every agent action passes through the fence
2. **Risk Scoring** - Automatic assessment of potential damage
3. **Instant Kill** - Emergency stop in under 100ms
4. **Audit Trail** - Complete log of what was attempted vs allowed
5. **Alerts** - Email/SMS when suspicious activity occurs

## How It Works

Wrap your AI agent with Runtime Fence:

```python
from runtime_fence import RuntimeFence, FenceConfig

fence = RuntimeFence(FenceConfig(
    agent_id="my-coding-assistant",
    blocked_actions=["delete", "exec", "rm"],
    blocked_targets=["production", ".env", "api_keys"],
    spending_limit=50.0
))

# Validate before allowing
result = fence.validate("delete", "user_data.sql")
if result.allowed:
    # Proceed
else:
    print(f"Blocked: {result.reasons}")
```
